Dynamic Chess Rules

Objective

The objective of Dynamic Chess is the same as standard chess: to checkmate your opponent's king.

Gameplay

The game follows the standard rules of chess, with the following exceptions:

  • Piece Movement

    Pieces can be moved using the mouse or the keyboard.

    Mouse

    Click on a piece to select it, then click on a valid cell to move it.

    Keyboard

    Press a key to select a piece type (p, r, n, b, q, k). If multiple pieces of the same type can be moved, pressing the same key again will cycle through the available pieces.

    Use the numpad (1-9) to move the selected piece. The direction is based on the numpad layout.

    For pawns, rooks, bishops, and queens, the distance of the move is determined by how long you press the key.

    For knights, a short press on a numpad key will move the knight to the first possible cell in that direction, and a long press will move it to the second possible cell.

  • Cooldowns

    Each piece has a cooldown period after it moves, during which it cannot be moved again. The cooldown is represented by a grey overlay on the piece's square, which fades as the cooldown expires.

    • Pawns: 2 seconds
    • Knights: 6 seconds
    • Bishops: 6 seconds
    • Rooks: 9 seconds
